硬件感知调度与在线校准 — 视觉化卡片（ljg-card 风格）

**概念压缩**：DSpark 通过置信度头 + 吞吐量参考曲线动态匹配最优验证长度，用顺序温度缩放将校准误差从 3%-8% 压到 ~1%，整套调度逻辑完全在 GPU 内部执行。

# 硬件感知调度核心（3 步）
def schedule(conf_head, throughput, draft_hidden, batch_size):
    # 1. 预估存活概率
    probs = conf_head(draft_hidden)  # [K]
    # 2. 计算净收益
    gains = [(probs[:K].sum() + 1) / throughput.query(B, K)
             for K in range(1, K_max+1)]
    # 3. 选择最优 K
    return np.argmax(gains) + 1

# 顺序温度缩放核心（1 行）
